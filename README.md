# PANDIApy
pandia-py is a lightweight offshoot of [mooonpy](https://github.com/jdkemppa/mooonpy), which is itself a project based on [LUNAR](https://github.com/CMMRLab/LUNAR). The goal of this project is to create a minimal version of mooonpy that is more willing to lean on 
3rd party libraries.
